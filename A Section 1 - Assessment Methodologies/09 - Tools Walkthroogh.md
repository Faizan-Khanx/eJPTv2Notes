
## [VILLAIN](https://github.com/t3l3machus/villain)

**Villain**Â is a C2 framework for managing multiple TCP sockets and HoaxShell-based reverse shells, offering enhanced functionality and inter-instance sharing.

### Key Features:

- **Payload Generation**: Create customizable payloads for Windows and Linux.
- **Dynamic Shell**: Switch between shell sessions easily.
- **File Uploads**: Upload files via HTTP.
- **Auto-HTTP Requests**: Execute scripts against sessions (unstable).
- **ConPtyShell**: Gain fully interactive Windows shells.
- **Team Chat**: Built-in communication for teams.
- **Session Defender**: Prevents command errors that could hang shells.
